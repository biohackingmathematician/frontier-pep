## Evidence Tier Definitions

| Tier | Confidence | Description |
|------|------------|-------------|
| 1 | 1.0 | Regulatory approval |
| 2 | 0.85 | Late clinical trials (Phase II/III) |
| 3 | 0.65 | Early clinical trials |
| 4 | 0.45 | Preclinical only |
| 5 | 0.25 | Mechanistic/in vitro |
| 6 | 0.15 | Anecdotal |
